## 👨‍💻 À propos de moi

Intégrateur Web chez **Mediactive Digital**, basé en **Picardie, France**. Je crois profondément que le développement web, c'est avant tout une affaire humaine : écouter, comprendre, et trouver la bonne solution — même quand le problème est tordu.

- 🏠 **Remote** — Je travaille depuis chez moi
- 🤝 Toujours disponible pour aider, débloquer une situation ou partager une solution, sans jugement
- 💡 J'aime les défis techniques complexes autant que les échanges bienveillants avec mes collègues et la communauté
- 🌱 Convaincu qu'on apprend mieux ensemble qu'en solo
